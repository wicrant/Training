Folder for Python programs made during Udemy Training

Steps to add and commit a file

1) git clone https://github.com/wicrant/Training.git

2) git add readme.txt

3) git commit -m "First Commit"

4) git branch -M main

5) git push -u origin main      #Enter login and password when prompted

Steps to pull a file from a repository

1) git fetch origin

2) git pull origin

Use git remote -v to see a list of remotes
